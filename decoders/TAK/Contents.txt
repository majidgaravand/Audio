Final release of TAK 2.1.0 ((T)om's lossless (A)udio (K)ompressor)

It consists of:

- TAK Applications 2.1.0 in "\Applications".
- TAK Winamp plugin 2.1.0 in "\WinAmp".
- TAK Software Development Kit 1.1.1 in "\SDK".
- TAK Decoding library 2.1.0 in "\Deco_Lib".

Each folder contains a Readme file with licence and other important
information.
